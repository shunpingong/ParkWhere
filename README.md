# ParkWhere

## Getting Started

1. Install the required dependencies

   ```bash
   npm install
   ```

2. Run the app

   ```bash
   npm start
   ```

3. Login with dummy account
   ```bash
   Email: test@account.com
   Pass: 123456
   ```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
